Apa itu web Developer

Web developer bekerja secara independen sebagai freelancer atau dengan perusahaan tim untuk membuat situs web. Tergantung pada pekerjaan, profesional ini mungkin fokus pada pengembangan front-end, yang melibatkan perancangan situs dan memproduksi konten, atau pengembangan back-end, yang melibatkan menulis kode untuk membuat fitur situs web berfungsi.

Apa yang web developer lakukan

Pengembangan web dan desain web memiliki banyak kesamaan, tetapi mereka berbeda di beberapa bidang utama. Desain web berkaitan dengan front-end dari sebuah situs web, mengenai dirinya dengan penampilan dan pengalaman pengguna. , di sisi lain, berkaitan dengan membangun front-end teknis dan / atau kode back-end yang menginformasikan fungsi situs. Pengembang web dapat bekerja secara eksklusif pada kode front-end, kode back-end, atau keduanya, dan banyak pekerjaan modern juga memerlukan tumpang tindih ke dalam desain web. Para profesional ini dapat bekerja secara independen secara freelance atau dengan departemen pemasaran atau TI.

Dasar yang harus dipelajari web developer



HTML

Bahasa markup yang digunakan untuk menyediakan struktur untuk konten di web, HTML5 adalah versi terbaru dari standar HTML. Pengembang web menggunakan HTML5 untuk membentuk aturan dasar halaman web, termasuk apakah akan memformat konten sebagai paragraf, judul, daftar, tautan, atau jenis elemen lainnya.

CSS

Pengembang web menggunakan CSS untuk mengontrol tata letak dan tampilan elemen di beberapa halaman situs. Bahasa CSS dapat mempengaruhi presentasi wajah font, warna, spasi di antara elemen, dan ukuran elemen pada halaman.

Javascript

JavaScript digunakan di sebagian besar situs web modern, dan ini adalah bahasa pemrograman landasan untuk halaman web interaktif. Pengembang web menggunakan JavaScript untuk memberikan "kehidupan" ke halaman, karena mengontrol perilaku seperti pemuatan dinamis. Bahasa ini juga menentukan respons elemen terhadap klik pengguna, menciptakan kemungkinan interaktif yang tak terhitung jumlahnya.
